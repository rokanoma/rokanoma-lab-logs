## About

データアナリティクスエンジニアとして、データモデル設計・パイプライン構築・BIレポート作成を主軸に業務を行っています。

一方で、社内ポータルの立案から実装・運用まで一人で手がけるなど、必要とあれば領域を問わず動いてきました。

「誰かがやるだろう」ではなく「自分がやる」を選んできた人間です。

## 実績

### NotesアーカイブのSharePoint移植

- Notes運用終了（Sunset）に伴い、25DB分の閲覧専用ページをSharePointへ移植
- HTML / CSS / JavaScriptで元のNotesUIを再現した閲覧専用アーカイブを設計・実装
- 自部門分を自発的に引き受け、IT部門への外注費用削減に貢献（削減額：約500万円相当）

### 社内ポータルサイト構築・運用

- 業務部門向けSharePointチームサイトを立案・企画・要件定義・設計・実装・運用保守まで一貫して担当
- 問い合わせ対応・ナレッジ共有・ツール提供など多目的ポータルとして運用
- サブサイトは年次集中期に1日100件超のアクセスをさばく実績あり

## 現在開発中：ポートフォリオサイト「Rokanoma」

- 自己紹介兼実験場として個人開発中（rokanoma.com）
- AIキャラクター「ロカノマ」がサイト常駐し、訪問者の案内・問い合わせ対応を担う設計
- 右下に2頭身キャラクターとして常駐し、クリックでサイト案内・自己紹介・仕事依頼へ誘導
- 将来的にはGemini APIで「自分の分身」として機能させ、クライアントとの初期接点を担う秘書的役割を想定

**技術構成**

HTML / CSS / JS + FastAPI + LangChain + ChromaDB + Docker

**コンセプト**

作りながら壊せる最短の本番環境。「やってみた」を積み重ねる場所。
